üìù Exam Conduction Module

This module presents 15 multiple-choice questions to the user. Each question is displayed with four options, and the user's answers are collected. A 5-minute timer, implemented using Python's time module, ensures the exam is time-bound. The module uses basic Python structures like lists and dictionaries and is fully compatible with online compilers.

üìä Result Evaluation Module

After the exam, this module compares the user's answers with the correct ones, calculates the total score, computes the percentage, and provides personalized feedback such as "Excellent", "Good Job", or "Needs Practice". If the time limit is exceeded, it auto-submits the test and displays the results instantly.
